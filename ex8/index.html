Link Externo

Sintaxe para adicionar link externo a um texto:

<a href="link" target ="_blank" rel="external"> descrição ou texto que sera clicado para direcionar ao link </a>

Em target, pode ser adicionado um parametro que determina em qual janela será carregado o link.
_self para carregar na janela atual, e _blank para abrir uma nova janela.

Link Interno

Para usar o link interno a sintaxe é a mesma, somento mundando a referencia, que no caso dos links externos
são as URL, já nos internos são o nome do arquivo. Outra mudança é o rel, que pode ser prev ou next para indicar
se o link é para uma pagina anterior ou uma proxima pagina.

obs: Se alguma pagina estiver em uma sub-pasta da pasta onde está o index, deve-se passar
o caminho completo.

por exemplo: sub_pasta/pag.html.

Já no arquivo da pagina que está em outra pasta deve-se adicionar no caminho o seguinte "../"
antes do index, isso serve para volta para a pasta principal, dependendo da orgaziação
das pastas cada "../" volta uma pasta.

ex: <a href="../index.html"> </a>

Sintaxe para link que carrega a proxima pagina:

<a href="nome do arquivo da pagina" target ="_self" rel="prev ou next" > descrição ou texto que sera clicado para direcionar ao link </a>

Links para Download

Para fazer um link de download, a sintaxe continua a mesma. A mudança ocorre no href onde ao invés de colocar uma url ou caminho para um arquivo html, adiciona-se o caminho para o arquivo que deseja fazer o download.

Sintaxe para link de download:

<a href="caminho do arquivo" download = "caminho do arquivo" type = "tipo do arquivo" > descrição ou texto que sera clicado para direcionar ao download </a>
