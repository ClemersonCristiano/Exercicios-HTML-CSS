Exercicio 5

Utilizando tags semanticas para formatar texto, tag <mark> para marcar um texto isso pode ser feito,
criando uma tag <style> e passando mark como parametro.
<style>mark{background-color: rgb(5, 255, 43);} </style>

tag <strong> para deixar o texto em negrito, e tag <em> para deixar em italico.
