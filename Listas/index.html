Criando Listas Utilizando tags <ol>, <ul> e <li>.

Estrutura de uma lista ordenada: <ol>

                    
                        <ol>
                            <li> </li>
                            <li> </li>
                            <li> </li>
                            <li> </li>
                            <li> </li>
                        </ol>

Estrutura de uma lista não ordenada <ul>

                    
                        <ul>
                            <li> </li>
                            <li> </li>
                            <li> </li>
                            <li> </li>
                            <li> </li>
                        </ul>

Estrutura do Item da Lista: <li>

                    
                        <li> </li>
                        <li> </li>
                        <li> </li>
                        <li> </li>
                        <li> </li>

Exercicio de lista
