Formatando codigo-fonte com Tags <pre> e <code>

Para a formatação ficar como está abaixo, foram utilizadas as tags <pre> e <code>. A tag <pre>
serve para deixar o texto no formato que esta escrito.
Já a tag <code>, serve para deixar os caracteres do texto no mesmo espaçamento, como acontece
em um trecho de código. para fazer isso a tag <code> deve esta dentro da tag <pre>.

Disposição das tags

                    
                        <pre>
                    
                            <code>

                            </code>
                    
                        </pre>

Exemplo de Codigo em C para calcular media de três numeros
formatado utilizando as tags. 

                    
    
                        #include 
    
                            int calcular_media(int n1, int n2, int n3)
                            {
                                int result = 0;
                                return result = (n1 + n2 + n3) / 3;
                            }
    
                            int main ()
                            {
    
                                int n1 = 5, n2 = 10, n3 = 8;
    
                                printf("A media dos numeros %d + %d + %d e: %d", n1, n2, n3, calcular_media(n1, n2, n3) );
    
                                return 0;
                            }
